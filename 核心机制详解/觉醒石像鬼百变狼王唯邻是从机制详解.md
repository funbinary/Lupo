# 觉醒石像鬼百变狼王唯邻是从机制详解

## 概述

觉醒石像鬼、百变狼王与唯邻是从（厄运傀儡）是狼人阵营中三种特殊角色机制。

- **觉醒石像鬼**：狼队首夜选择一名相邻玩家转化为狼，技能正常生效，该玩家知道自己变为狼
- **百变狼王**：开局从神牌池随机抽取身份，技能正常生效
- **唯邻是从（厄运傀儡）**：狼队首夜选择一名相邻玩家转化为狼，但该玩家不知情，大部分技能失效或反转

三者均不与狼队见面。

## 转化方式

| 机制    | 转化时机 | 选择方式       | 被转化者知情 |
| ----- | ---- | ---------- | ------ |
| 觉醒石像鬼 | 第一夜  | 狼队选择一名相邻玩家 | 是      |
| 百变狼王  | 开局   | 从神牌池随机抽取   | 是      |
| 厄运傀儡  | 第一夜  | 狼队选择一名相邻玩家 | 否      |
