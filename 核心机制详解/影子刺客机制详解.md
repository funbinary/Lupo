# 影子刺客机制详解

## 角色介绍

### 影子

- 开局选择一名玩家作为主人
- 每天有睁眼轮次，若主人前一天倒牌，获知主人身份并继承其身份技能
- 与主人占同一种坑位，继承后不与狼队见面
- **仅继承狼人身份后可被查验出狼**

### 刺客

- 第一天最后睁眼
- 阵营与影子主人对立（好人→帮狼，狼人→帮好人）
- **帮狼人玩时验出来是狼，不与狼队见面**

***

## 技能介绍

### 影子

| 阶段 | 效果                     |
| -- | ---------------------- |
| 开局 | 选择主人                   |
| 日常 | 每天睁眼，主人倒牌后获知身份并继承技能    |
| 继承 | 获得主人全部能力（女巫药、魔术师换牌次数等） |

- 主人出局后变双边阵营（好人时为神/民）
- 绑定刺客则成第三方，第二晚起每晚各刀一人

### 刺客

| 技能   | 效果                      |
| ---- | ----------------------- |
| 阵营判定 | 与影子主人阵营对立               |
| 死亡刺杀 | 死后可刺杀一名玩家，对立阵营倒牌，同阵营不倒牌 |

- 刺中影子则刺客单独胜利

***

## 胜负判定

| 情况   | 胜利条件              |
| ---- | ----------------- |
| 常规   | 随各自阵营获胜           |
| 第三方  | 影子+刺客独立阵营，完成屠城/屠边 |
| 刺客特殊 | 刺中影子 → 刺客单独胜利     |
