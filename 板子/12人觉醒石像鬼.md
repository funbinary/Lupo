# 12人觉醒石像鬼

## 板子角色

**好人阵营（5神+4民）：**

- 预言家
- 女巫
- 猎人
- 猎魔人
- 摄梦人
- 4平民

**狼人阵营（4狼）：**

- 觉醒石像鬼（百变狼王）
- 3小狼

**特殊说明：** 觉醒石像鬼首夜在狼队行动之前选择模仿一张神职牌获得对应技能。若不使用百变技能，次夜自动获得狼王技能。

- 当转化的狼为平民时，猎魔人归入民坑

***

## 技能介绍

### 预言家

每晚可查验一名玩家阵营，得知"好人"或"狼人"。

### 女巫

拥有1瓶解药和1瓶毒药，每晚可选择使用其中1瓶（不可同时使用），整局各限1次。

- 解药：可救活当晚被狼刀的玩家
- 毒药：可毒杀任意一名玩家

### 猎人

死亡时可选择开枪带走一名玩家。被毒死、被梦死时无法发动技能。

### 猎魔人

从第二夜开始，每晚可选择一名玩家进行狩猎。

- 若目标为狼人，则目标死亡
- 若目标为好人，则猎魔人自身死亡

### 摄梦人

每晚必须选择一名玩家成为"梦游者"。

- 被摄梦者当晚免疫狼人袭击、女巫毒药
- 连续两晚摄梦同一人，该玩家死亡（梦游死）
- 摄梦人夜间出局，被摄梦连锁效果立即触发

### 觉醒石像鬼

| 模仿角色 | 获得的技能                      |
| ---- | -------------------------- |
| 预言家  | 每晚可查验一名玩家具体身份              |
| 女巫   | 获得一瓶毒药（无解药）                |
| 猎人   | 死亡时可开枪（被毒死、被猎魔人猎死、梦游死不能开枪） |
| 猎魔人  | 可狩猎一名玩家，狩猎到神或者狼自己死，狩猎平民死   |
| 摄梦人  | 与摄梦人一样                     |

### 小狼

每晚与其他狼人见面，共同选择一名玩家进行袭击。

***

## 带刀顺序

1. 小狼共同带刀
2. 3小狼全部出局后，觉醒石像鬼带刀

***

## 首夜行动顺序

1. 狼人出 - 选择转化者
2. 狼人：选择刀谁
3. 叫下转化者
4. **摄梦人** - 选择梦游者
5. **女巫** - 使用解药/毒药
6. **猎魔人** -认识一下
7. **猎人** - 确认是否死亡及能否开枪
8. **预言家** - 查验玩家身份

## 次夜行动顺序

1. 转化者：带刀状态
2. 狼人：选择刀谁
3. **摄梦人** - 选择梦游者
4. **女巫** - 使用解药/毒药
5. **猎魔人** - 是否发动技能
6. **猎人** - 确认是否死亡及能否开枪
7. **预言家** - 查验玩家身份
