# 12人机械狼

## 板子角色

**好人阵营（4神+4民）：**

- 通灵师
- 女巫
- 猎人
- 守卫
- 4平民

**狼人阵营（4狼）：**

- 机械狼
- 3小狼

**特殊说明：** 机械狼可在任意一夜开启"学习"状态，选择学习场上一名玩家的身份及技能。整场游戏只能学习一次。

***

## 技能介绍

### 通灵师

每晚可查验一名玩家身份，得知其具体身份（预言家/女巫/猎人/守卫/平民/狼人等）。

- 查验未学习前的机械狼显示为特殊手势（双手狼爪）
- 查验学习后的机械狼显示为其学习对象的身份

### 女巫

拥有1瓶解药和1瓶毒药，每晚可选择使用其中1瓶（不可同时使用），整局各限1次。

- 解药：可救活当晚被狼刀的玩家
- 毒药：可毒杀任意一名玩家

### 猎人

死亡时可选择开枪带走一名玩家。被毒死时无法发动技能。

### 守卫

每晚可守护一名玩家（不能连续两晚守护同一人，可以空守）。

- 被守护的玩家免疫狼刀
- 若该玩家同时被女巫解救，则发生"奶穿"，该玩家死亡

### 机械狼

可在任意一夜选择学习一名玩家的技能：

| 学习对象 | 获得技能                            | 发动时机     |
| ---- | ------------------------------- | -------- |
| 通灵师  | 通灵师的验人                          | 从下一个夜晚开始 |
| 女巫   | 一瓶毒药                            | 从下一个夜晚开始 |
| 守卫   | 加强版守护（可抵挡狼刀、毒药、猎人晚上子弹），但是只能生效一次 | 从下一个夜晚开始 |
| 猎人   | 猎枪                              | 从当晚开始    |
| 平民   | 被验是金水                           | 从当晚开始    |
| 狼人   | 额外一刀（与其他狼人刀同一人可刀穿）              | 3狼出局后的夜晚 |

- 与其他狼人晚上互相不见面
- 白天不能自爆
- 3小狼全部出局后带刀

### 小狼

每晚与其他小狼见面，共同选择一名玩家进行袭击。

***

## 带刀顺序

1. 3小狼共同带刀
2. 3小狼全部出局后，机械狼带刀

***

## 首夜行动顺序

1. **机械狼** - 确认是否学习技能（首夜）
2. **守卫** - 选择守护目标
3. **狼人见面** - 小狼见面，确认袭击目标
4. **女巫** - 使用解药/毒药
5. **猎人** - 确认是否吃毒及能否开枪
6. **通灵师** - 查验玩家身份
7. **机械狼** - 发动学习获得的技能

## 次夜行动顺序

1. **机械狼** - 带刀状态
2. **守卫** - 选择守护目标
3. **狼人见面** - 小狼见面，确认袭击目标
4. **女巫** - 使用解药/毒药
5. **猎人** - 确认是否能否开枪
6. **通灵师** - 查验玩家身份
7. **机械狼** - 发动学习获得的技能
